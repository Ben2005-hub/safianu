WELCOME TO MY HOMEPAGE


For the first time I was not familiar with anything concerning web development. But we were introduced to a free web development site by my tutor Mr. Danso. The name of the website is freecodecamp. An interactive web design platform which helps students to learn from scatch. On the first day it was very hard because I had no idea about html, css and all those stuffs. Through this platform i was able to complete several tasks such as a CatPhotoApp, SurveyApp and many more. Through that i was able to develop several websites on the platform. This earned me several certificates. Some of the certificates includes SurveyApp which i was able to complete on my own through the sites help. After building about five certificate Applications, the tutor asked us to host the website we have created. With that it was very hard because I was new to everything about hosting. Many students were sharing their work through hosting sites such as tiiny.site, github.com and many more.

FreeCodeCamp is a nonprofit organization that provides a free, self-paced coding curriculum to help learners develop software development skills.

The platform's supportive online community, along with its wealth of resources, has helped millions of learners worldwide embark on new careers in technology.
